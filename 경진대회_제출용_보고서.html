트루부산(TrueBusan) MVP 개발 완료 보고서

1. 서비스 개요 및 MVP 배포 주소

서비스명: 트루부산 (TrueBusan) - AI 및 공공데이터 기반 부산 안심 관광/미식 큐레이션

사용자용 웹 서비스 (MVP): https://truebusan-frontend.vercel.app

관리자용 대시보드: https://truebusan-frontend.vercel.app/admin (ID: admin / PW: 1234)

개발 환경: Next.js (Frontend), FastAPI (Backend), Vercel (Cloud Hosting)

2. 주요 기능 및 UI 스냅샷

2.1 사용자 맵 대시보드 및 안심 큐레이션 UI

카카오 지도 API를 활용하여 부산 주요 지역(해운대, 광안리 등)의 실제 핫플레이스를 렌더링하고, 장소 클릭 시 직관적인 사진과 안심 데이터가 담긴 바텀시트를 제공합니다.

[ 📷 스크린샷 삽입 구역 ] truebusan-frontend.vercel.app 접속 후 메인 지도 화면 캡처

2.2 AI 기반 드래그 앤 드롭(DnD) 동선 추천 시스템

사용자가 선택한 장소들을 기반으로 최적의 여행 코스를 생성하며, 드래그 앤 드롭 인터페이스를 통해 직관적으로 동선 순서를 변경할 수 있습니다.

[ 📷 스크린샷 삽입 구역 ] 우측(또는 하단)에 열리는 '동선 추천(코스 생성)' UI 캡처

3. 지능형 데이터 파이프라인 (Data Pipeline)

단순 데이터 수집을 넘어, 서버 부하와 AI API(LLM) 호출 비용을 획기적으로 절감하는 '30/40/30 AI 분석 전략 알고리즘'을 자체 설계하여 파이프라인에 적용했습니다.

  • 상위 30% (High Confidence): 긍정 리뷰 압도적 다수. 정밀 AI 분석을 거쳐 자동 PASS(안심) 처리.
  • 하위 30% (Low Confidence): 부정 리뷰(바가지, 불친절 등) 다수 발생. 정밀 AI 분석 후 자동 FAIL(경고 및 리스트 제외) 처리.
  • 중간 40% (Neutral): AI API 토큰 소모를 방지하기 위해 분석을 스킵하고 기본 장소 정보만 구성하여 운영 리소스 극대화.
[ 📷 스크린샷 삽입 구역 ] 파이썬 크롤러(crawler.py) 실행 로그 화면 캡처

4. 관리자 백오피스 및 공공데이터 융합 (Data Management)

수집된 데이터의 객관성과 신뢰도를 보장하기 위해 지자체 공공데이터를 융합한 백오피스(관리자 대시보드)를 구축했습니다. 관리자는 직관적인 UI를 통해 안심 마크를 직접 제어하고 검증 내역을 관리할 수 있습니다.

검증 기준 (공공데이터) 데이터 적용 설명 AI Trust Score 반영
지자체 안심식당 부산시/구청에서 인증한 위생 준수 식당 데이터를 크롤링 데이터와 교차 검증 + 가중치 부여
착한가격업소 행정안전부 지정, 바가지 요금 없는 정찰제 식당 목록 검증 + 강력한 가중치
식약처 위생등급 '매우 우수', '우수' 등급 취득 여부 시스템 자동 연동 + 가중치 부여
[ 📷 스크린샷 삽입 구역 ] /admin 접속 후 '리뷰 & 안심 검증' 탭의 데이터 테이블 캡처

5. 기대 효과 및 향후 발전 방향

본 MVP는 단순한 장소 나열식 관광 앱을 넘어, '바가지요금'과 '불친절'을 기술적으로 필터링하는 '신뢰 기반의 관광 생태계'를 제안합니다. 향후 실시간 리뷰 크롤링을 클라우드 스케줄러(Cron)로 100% 자동화하고, 외국인 관광객을 위한 다국어(영어, 중국어, 일본어) AI 자동 번역 지원을 통해 부산의 글로벌 스마트 관광 경쟁력 강화에 기여할 것입니다.
